Display list of posts, pages or any custom post types anywhere using block and widget. Show as grid, list or posts slider.

== Description ==
Display post types provides easiest way to fetch, sort and display list of posts, pages or any custom post types using block or widget. Create beautiful & responsive grids, lists or sliders of your fetched content.

= What this plugin offers =

* Fetch any post, page or post types using available filters and sort your selection.
* Choose what content items to be displayed (from available options)
* Choose a style to display your content. You can choose from list, grid or slider views. Variuos customization options
* Live preview of your changes in the display post types block.

= Setup Gallery Widget =

Display post types in any widget area of your website.

1. After activating the plugin, visit Appearance > Widgets in admin dashboard.
1. Look for 'Display Post Types' widget in left 'Available Widgets' section.
1. Drag the widget to any available sidebar/widget area.
1. Select a post type to be displayed.
1. Select various filter, sort and styling options to make a unique layout.
1. Click [Save] button.

= Setup Gallery Block =

Display post types list on any post or page using editor block. Make sure you have not disabled WordPress latest block editor.

Setup

1. After activating the plugin, visit any post or page's edit screen.
1. In main content area, click on '+' icon to add a new block.
1. Search for 'Display Post Types' block or Select from "Layout Elements > Display Post Types".
1. Select a post type to be displayed.
1. Select various filter, sort and styling options to make a unique layout.
1. Click [Publish] or [Update] button.

== Installation ==

Installing “Display Post Types” can be done either by searching for “Display Post Types” via the “Plugins > Add New” screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the ‘Plugins > Add New > Upload’ in your WordPress dashboard
1. Activate the plugin through the ‘Plugins’ menu in WordPress
